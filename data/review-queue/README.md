# Review Queue

Drafts generated by the sensing-layer scripts land here for human review.

## Structure

- `f2/`: Startup thesis drafts (one per pillar per run)
- `f4/`: Post-event recap drafts (recap, notes, field-note outline)

## How to approve

Reply "approve <filename>" to the Telegram bot. The bot moves the file to
`data/approved/` and logs the approval.

## Important

Nothing in this directory is published automatically. Every draft requires
explicit human approval before it reaches any public surface.
